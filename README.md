# Pioneer Online — Telegram Bot

Официальный тренировочный ассистент зала **Пионер**.

> Зал Пионер работает с 2014 года. Более 2000 человек достигли результатов — от первых успехов в фитнесе до разрядов мастера спорта. Pioneer Online — цифровое продолжение этой экспертизы.

---

## Быстрый старт — 4 шага

### Шаг 1. Создай Telegram-бота (2 минуты)

1. Открой Telegram, найди @BotFather
2. Напиши `/newbot`
3. Придумай имя: например `TrainBot Парфенов`
4. Придумай username (должен заканчиваться на `bot`): например `parfenov_train_bot`
5. BotFather пришлёт **токен** вида `1234567890:ABCdef...` — скопируй его

### Шаг 2. Узнай свой Telegram ID

1. Найди бота @userinfobot в Telegram
2. Напиши ему `/start`
3. Он покажет твой `id` — это число, скопируй его

### Шаг 3. Настрой файл .env

```bash
cp .env.example .env
```

Открой `.env` и заполни:
```
TELEGRAM_TOKEN=токен_от_BotFather
ANTHROPIC_KEY=sk-ant-api03-...  (тот же ключ что в TrainBot)
ADMIN_ID=твой_telegram_id
```

### Шаг 4. Запусти бота

**Вариант А — на своём компьютере (для теста):**

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить
python run.py
```

Бот работает пока открыт терминал. Нажми Ctrl+C чтобы остановить.

**Вариант Б — на сервере Railway.app (бесплатно, работает 24/7):**

Смотри раздел "Деплой на Railway" ниже.

## Деплой на Railway.app (бесплатно, 24/7)

Railway даёт $5 кредитов в месяц бесплатно — этого хватает для бота.

1. Зарегистрируйся на [railway.app](https://railway.app) через GitHub
2. Нажми **New Project → Deploy from GitHub repo**
3. Загрузи папку `trainbot_telegram` как репозиторий GitHub:
   ```bash
   git init
   git add .
   git commit -m "TrainBot init"
   # создай репозиторий на github.com и запушь
   git remote add origin https://github.com/ТВО_ИМЯ/trainbot.git
   git push -u origin main
   ```
4. В Railway выбери этот репозиторий
5. В разделе **Variables** добавь переменные:
   - `TELEGRAM_TOKEN` = токен бота
   - `ANTHROPIC_KEY` = ключ Anthropic
   - `ADMIN_ID` = твой Telegram ID
6. Railway автоматически найдёт `Procfile` и запустит бота
7. Готово — бот работает 24/7!

## Альтернатива: Render.com (тоже бесплатно)

1. Зарегистрируйся на [render.com](https://render.com)
2. New → **Background Worker**
3. Подключи GitHub репозиторий
4. Build Command: `pip install -r requirements.txt`
5. Start Command: `python run.py`
6. Добавь Environment Variables (те же три переменные)
7. Deploy!

## Что умеет бот

**Клиенты пишут:**
- Проходят 6-шаговую анкету
- Получают персональную программу по неделям
- Пишут "далее" — получают следующую неделю
- История сохраняется между сессиями

**Ты как тренер:**
- `/clients` — список всех клиентов (только для ADMIN_ID)
- Каждый клиент изолирован — видит только свои данные
- История хранится в `clients.json` на сервере

**Команды для клиентов:**
- `/start` — начать / перезапустить анкету
- `/reset` — очистить историю и начать заново
- `/help` — справка

## Вопросы и проблемы

**Бот не отвечает?**
- Проверь что `TELEGRAM_TOKEN` правильный
- Проверь что бот запущен (логи в терминале или Railway)

**Ошибка Anthropic API?**
- Проверь баланс на console.anthropic.com
- Проверь что `ANTHROPIC_KEY` начинается с `sk-ant-`

**Хочу ограничить доступ — только мои клиенты?**
- Добавь список разрешённых user_id в `.env` как `ALLOWED_USERS=123,456,789`
- Напиши мне — добавлю эту функцию в код
